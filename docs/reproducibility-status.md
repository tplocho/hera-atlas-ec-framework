# Reproducibility Status

## Current state
This repository contains the code layer, but not yet a fully stabilized publication companion package.

## Presently included
- source MATLAB code for the main framework components
- top-level documentation of scope and release posture

## Presently not included
- manuscript draft PDF
- internal audit PDFs
- submission-sensitive figures
- a finalized example dataset bundle for ATLAS inputs

## Honest interpretation
The code is real and substantial. The current limitation is packaging completeness, not absence of technical content.

## Recommended next reproducibility steps
1. standardize naming across source files and interface text
2. decide whether to ship generated example CSV datasets or a documented generation path
3. add one small, stable example workflow
4. publish a post-submission figure subset if and when safe
