# Repository Scope

This repository contains the software and supporting notes for the HERA–ATLAS framework.

## Included
- MATLAB source code for HERA, ATLAS, and the ATLAS interactive configuration layer
- concise documentation describing the framework structure and execution flow
- citation metadata and license information

## Maintained outside the repository
- manuscript drafts
- internal review or audit documents
- selected publication figures and layout assets
- project-local working material that is not part of the software release

## Practical intent
The repository is intended to present the framework as a clean codebase for inspection, discussion, and future extension.
