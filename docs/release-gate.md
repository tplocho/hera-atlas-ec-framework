# Release Gate

This repository is close to publishable as a public code asset, but the following issues should still be acknowledged before treating it as fully mature:

## 1. Reproducibility gap
`ATLAS.m` expects generated CSV datasets that are not currently bundled in this public-safe cut. The code is still useful and real, but the repo does not yet provide a fully frozen end-to-end reproduction package.

## 2. Figure policy
Submission-sensitive figures are intentionally excluded for now because the publication package may still change.

## 3. Manuscript separation
Draft manuscript files and internal audits are intentionally excluded from the public repository.

## 4. Interface hardening still desirable
The public-facing naming has been normalized in this scaffold, but the repository would still benefit from a fuller worked example, a clearer dataset-generation path, and post-submission curation of a stable example-output set.
